Spielort
========


Die Örtlichkeit, an dem
 [Veranstaltungen](veranstaltung.md "Veranstaltung") wie Konzerte stattfinden. Zum Beispiel eine Diskothek, ein Konzerthaus oder aber auch eine Stadt.



## Felder {.fs-3 .mt-5 .border-bottom}


### Name {.fs-5 .mt-5}



Name der Örtlichkeit. Also der Name des Konzerthauses, aber auch Name der Stadt/Region (beispielsweise: "Moers" als Name für den Spielort des Moers Festivals).

### Ort {.fs-5 .mt-5}



Angabe zur regionalen Zuordnung des Spielortes.

### Beschreibung {.fs-5 .mt-5}



Ein Feld für weitere Angaben, welche in kein anderes der Felder passen.

### Bemerkungen {.fs-5 .mt-5}



Notizen für Archiv-Mitarbeiter. Nur Mitarbeiter können dieses Felder sehen.



---


### Alias {.fs-5 .mt-5}



Weitere Namen unter denen der Spielort bekannt ist.
