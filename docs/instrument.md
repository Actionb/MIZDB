Instrument
==========


Die Datensätze in dieser Kategorie werden dazu benutzt, um Angaben zu Instrumenten bei Musikern zu machen.



## Felder


### Instrument { .fs-5 }



Der Name des Instruments.

### Kürzel { .fs-5 }



Abkürzung des Instruments, wie sie in Notenschriften oder als Angabe bei Live-Auftritten verwendet werden. Siehe
 [Wikipedia: Abkürzungen Musikinstrumente](https://en.wikipedia.org/wiki/de:Abk%C3%BCrzungen/Musikinstrumente "wikipedia:de:Abkürzungen/Musikinstrumente").
