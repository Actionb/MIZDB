Bestand
=======


Jeder Bestand besteht aus einem [Lagerort](lagerort.md "Lagerort") und optional aus einer
 [Provenienz](provenienz.md "Provenienz"). Unter 'Anmerkungen' können weitere Angaben eingefügt werden; so zum Beispiel Vermerke über den Zustand bzw. der Beschaffenheit des Objektes. Jedem Bestand wird von der Datenbank eine einzigartige Signatur ([ID](id.md "ID")) zugewiesen.
