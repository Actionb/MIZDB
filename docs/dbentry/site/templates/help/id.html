ID

Die ID-Nummer eines Datensatzes findest du neben dem Namen der Kategorie:

Hinweis: Die Datenbank vergibt die IDs

Jeder Datensatz einer Tabelle besitzt genau eine ID. Und eine ID identifiziert auch genau einen Datensatz einer Tabelle. Dadurch spielt die ID für die Datenbank eine enorm wichtige Rolle und kann deshalb von Benutzern nicht verändert werden - sie wird stattdessen von der Datenbank selbst vergeben.

Abgesehen von der Suche nach IDs im Suchformular, wird die ID eines Datensatzes zur Kennzeichnung von Foto-Materialien verwendet: Foto#ID
