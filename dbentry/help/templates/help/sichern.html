Sichern

Es stehen drei Knöpfe zum Sichern eines Datensatzes zur Verfügung. Erst durch das Betätigen einer der Knöpfe werden die Daten des Formulars auch wirklich gespeichert. Verlasst ihr das Formular ohne eine Anfrage an den Server geschickt zu haben - z.B. wenn ihr einfach das Fenster schließt - können Änderungen nicht übernommen werden.

  • Sichern und neu hinzufügen
Das Formular wird zum Server geschickt, um den Datensatz zu speichern, und der Server liefert ein neues, leeres Formular zurück.
  • Sichern und weiter bearbeiten
Zwischenspeichern. Der Datensatz wird abgespeichert und das Formular wird zur weiteren Bearbeitung erneut dargestellt.
  • Sichern
Nachdem der Datensatz abgespeichert wurde, wird man zur Änderungsliste weitergeleitet.
